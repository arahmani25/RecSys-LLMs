Usability Tips & Setup

  • **Architecture**: The User Tower uses a simple Embedding lookup. The **Item Tower uses a Deep Learning MLP** (Genre Features → Hidden Layer → Embedding).
  • **Comparison**: The "Simple Embedding Rec. (Baseline)" uses the trained User Embedding ID lookup and the final Item Embeddings (derived from the MLP) for a comparison against the full deep model.
  • **Setup**: Ensure you have a `/data` folder containing `u.data` and `u.item`.
